Control de Texto con JavaScript

Tamaño de Fuente

Alineación del Texto

**Párrafo 1:** En este primer párrafo, vamos a comprobar cómo se aplica la funcionalidad de **cambio de tamaño de fuente**. Al presionar los botones de "Aumentar" o "Disminuir", el tamaño del texto dentro de este contenedor cambiará dinámicamente.

**Párrafo 2:** La segunda funcionalidad que probaremos es la **alineación del texto**. Los botones "Justificar" y "Alinear Izquierda" modificarán la propiedad CSS `text-align` de todo el bloque de texto.

**Párrafo 3:** Es importante notar que estamos siguiendo las indicaciones del ejercicio, que solicita implementar cada funcionalidad utilizando **funciones independientes**.

**Párrafo 4:** Este es el cuarto párrafo. La idea es que al modificar el estilo en el contenedor (`#text-container`), **todos los párrafos** hereden los cambios de tamaño y alineación automáticamente.
